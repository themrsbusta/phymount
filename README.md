Phymount is a lightweight, high-performance grid montage tool built with Python. Designed for speed and precision, it allows users to organize multiple photos into a single high-resolution grid without the need for complex editing software.

‚ú® Key Features
Smart Multi-Drop: Drag and drop a batch of photos (e.g., 6, 12, or more) directly into the app. It automatically populates empty slots in order.

Intuitive Controls:

Left Click + Drag: Pan and frame your photo within the slot.

Right Click + Drag: Swap photos between slots instantly (Reorganization mode).

Mouse Wheel: Smooth real-time Zoom in/out.

Professional Output: Export your montage in HD (720p), Full HD (1080p), or 4K (Ultra).

Edge Clamping: Intelligent logic ensures photos never "leak" or show the background, keeping your grid perfect.

üöÄ How to Use
Launch Phymount: Open the executable (now starts directly into the workspace).
